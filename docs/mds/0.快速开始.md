## 打开终端并进入框架根

```
> cd ./xlik
```

## 创建新项目

```
> xlik.exe new demo
```

## 测试

```
> xlik.exe run demo -l
```

## 使用历史工具管理项目

```text
你应该使用版本代码历史管理工具（Git、Svn等）来管理你的开发项目。
你可以使用第三方的托管平台（如GitHub、GitLab、Gtiee、Bitbucket），或自建（如Gogs、GitLab）管理工具，请至少在本地使用工具构建本地版本。
人无远虑必有近忧，没有人会知道自己什么时候会犯错。代码历史可让你查看每一次的修改，回滚版本、备份成果。
```
